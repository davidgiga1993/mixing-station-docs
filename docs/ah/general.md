# Allen & Heath

## Qu Connection modes

Mixing Station has two different modes for connection to the mixer:

- Qu-Drive enable
- Qu-Drive disable

The table below shows the differences between the two modes:

| Mode | Parallel instances | Qu-Drive access |
|------|--------------------|-----------------|
| On   | 2                  | Yes             |
| Off  | 7                  | No              |
